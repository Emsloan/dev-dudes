Dev Dudes is a two-person freelance web dev business: Elliott (technical) + Austin (client-facing).

**Austin** — the original. Been doing this longer (Squarespace sites, GoDaddy domain/DNS setup for clients). Gets clients through his personal manager (Austin has a creative career: VA, acting, music, modeling). Luxury sales background: Neiman Marcus, Suit Supply, high-end watches, jewelry, GoDaddy sales. Tastemaker. People person. The "rainmaker" of the duo.

**Elliott** — joined to raise the technical ceiling. Can build real hand-coded sites, Stripe integrations, custom everything — things Austin couldn't offer before.

**The pitch:** clients overpay for Squarespace/Shopify subscriptions forever. Dev Dudes builds it once, hand-coded, client owns it outright.

**Why:** Elliott saw a clear value opportunity in a post-Claude Code world where agentic dev makes custom sites fast and cheap to build, while SaaS website builders still charge monthly rent.
